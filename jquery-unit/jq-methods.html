  • Get () and Set (some value)
  • .text() is the same as textContent; it retrieves the text of an element. We can also pass in a value to change the text content.
  • .html() retrieves the content within the element. Passing in an value will allow you to change the html structure/contents.
  • attr() gets value of an attribute or set an attribute.
  • val() extracts value from input; set value for every matched element. Good for resetting a text input. Clears things up. Works on things with value inputs. So dropdown menus, checkboxes. etc.
  • addClass() adds class(es) to a selected element(s).
  • removeClass() removes a class from selected elements(s)
  • toggleClass() adds/removes a class(s) from selected element(s)
