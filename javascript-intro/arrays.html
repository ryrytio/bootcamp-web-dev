• Push and pop
    • Push means "to add" onto something. Like updating an array.
    • ARRAY.push("new item")
    • Pop means to "remove" the last element of an array
  • Shift and Unshift
    • Opposites of push and pop.
    • Unshift means to add to the front of array
    • Shift means to remove the first item in an array
  • IndexOf
    • Use to find the index of an item in an array
    • Value of "-1" means that it can't find this item in the array.
  • Slice
    • Use to copy parts of an array
    • ARRAY.slice(beginningIndex, endingIndex); not inclusive of last item, (it doesn't include the endingIndex).
    • .slice doesn't alter the original array.
    • ARRAY.slice() copies the whole array when fields are empty.
  • Nested Array
    • ARRAY[2][0] --> this means third item of the array, within that sub-array select the first item.
    • this is a 2 dimensional array bc an Array is within an Array.
